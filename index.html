CATEGORÍAS
⚠ LOS VÍDEOS NO SE REPRODUCIRÁN DESDE FILE://
El navegador bloquea YouTube, Instagram y otros embeds cuando el archivo se abre con doble clic.
Solución: Abre una ventana de CMD en la carpeta del archivo y ejecuta:
python -m http.server 8080
Luego abre: http://localhost:8080/Mis_cosas_63_v2.html
